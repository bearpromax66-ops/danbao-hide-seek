# 蛋宝捉迷藏

一个给孩子玩的键盘版卡通捉迷藏原型。无需鼠标、无需安装依赖。

## 启动

方式一：我已经在本机启动了预览服务，可以打开：

```text
http://127.0.0.1:4173/index.html
```

方式二：以后可以在终端运行：

```bash
python3 -m http.server 4173
```

然后打开同一个地址。

## 操作

- 菜单选择：`W/S` 或方向键
- 确认开始：`Enter`
- 移动：`W A S D` 或方向键
- 滚动加速：`Space`
- 躲藏者变身：`Q` 或 `Tab`
- 躲藏者指定形态：`1-0`
- 追捕者火眼金睛：`Command`
- 追捕者抓捕：碰到已经显形的躲藏者即可淘汰，`E` 也可以手动尝试
- 返回菜单：`Esc`

游戏进行中会自动锁定键盘输入并禁止选中文字。如果误切到其他窗口，点回游戏画面或按任意控制键即可继续。

## 玩法

- 一局 5 分钟，1 个追捕者，4 个躲藏者。
- 躲藏者是带小偷眼罩的小偷蛋宝，追捕者是带警察帽的警察蛋宝。
- 躲藏者可以在河边、公路、公园、广场变成 10 种卡通物体。
- 地图里会出现同类背景物，躲藏者可以混在合理位置的物体中伪装。
- 追捕者的火眼金睛可以无限使用，没有区域次数或冷却限制。
- 被火眼金睛扫到的躲藏者会变回本体，追捕者碰到显形躲藏者即可淘汰。
- 玩家作为躲藏者被淘汰后，会自动切到追捕者视角继续观战。
- 追捕者在倒计时内抓到全部躲藏者则胜利，否则躲藏者胜利。
